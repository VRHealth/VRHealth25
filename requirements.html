VRHealth Project Requirements

Project Specifications

Main Goal

Our goal is to continue work on the Virtual Reality (VR) Mental Health Platform by expanding its functions to providing activities within the developed environment to aid those suffering from Alheimer's Disease or dementia. The goal of these activities are related to retaining skills relating to visual perception and spatial awareness, thus these activities will focus on object localization. One of our primary goals will be to design object localization activities within the VR environment. A subgoal of this project to make the experience enjoyable for the user, with gamification of the activities being one method of execution. The expected tech stack will be Unity for development of the activities with the environment.


An example implementation would be a scavenger hunt, where a list of items is provided to the user. where the user is told to locate one item within an environment. Once they locate the item, they are told to locate a different item. After that, the user will be told to recall the location of an item they had previously found. This would repeat until either time runs out, or a round limit is reached. These items are visually distinct from one another and composed of everyday common objects. Visual and audio cues may be used to help the user find objects easier such as arrows, high contrast colors highlighting the object, or even music or sounds that increase in sound with increasing proximity to the target object.


Specific Goals

Two versions of an Object Localization Activity
Version 1
- Objects spawn anywhere randomly in a virtual environment
- User has to point out where each object was located
Version 2
- Objects at specific defined points, chosen at random in a virtual environment
- User must walk back to each object's location
2 virtual environments to perform the activity in
- A garden/park
- An apartment
5 unique items to locate per environment
- Each environment has its own set of items

Object Localization Activity

Version 1

  1. 1) Walk to the location of four objects
  2. 2) Walk to each location two more times
  3. 3) Using the Right Controller, point to the location of each object

Version 2

  1. 1) Walk to the location of one object
  2. 2) Either walk to the location of a different object OR Recall the location of a previous object
  3. 3) Repeat so many times until the activity is complete
